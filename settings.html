Set up the private sync worker

Deploy worker/baby-list-worker.js with the included Durable Object configuration, set a strong TOKEN secret and an exact ALLOWED_ORIGINS list, then paste its HTTPS URL and TOKEN here. Only the v3 worker is accepted.

The endpoint and token stay on this device. Normal snapshot links and backup files omit them.

Reset

The first button clears only packed status. A new-pregnancy reset also clears dates, item notes, tracker history, and emergency-card entries while keeping custom items, reminders, contacts, and sync settings.
